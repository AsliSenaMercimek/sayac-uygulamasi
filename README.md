#Sayaç Projesi

Basit ve kullanışlı bir sayaç uygulaması.  
HTML, CSS ve JavaScript kullanılarak geliştirilmiştir.

##Özellikler

- Artırma ( + ) butonu
- Azaltma ( − ) butonu
- Sıfırlama butonu
- Gerçek zamanlı değer güncelleme
- Mobil uyumlu arayüz

##Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
